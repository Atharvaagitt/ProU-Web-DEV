# Employee Task Tracker

A modern, responsive Employee Task Tracker dashboard built with React, Vite, and Tailwind CSS.

**[Live Demo](https://atharvaagitt.github.io/ProU-Web-DEV/)**

## Features

- **Dashboard**: Visual overview of task status with charts and stats.
- **Task Management**: Kanban-style board with drag-and-drop functionality.
- **Employee Directory**: Searchable list of employees with details.
- **Dark Mode**: Fully supported dark/light theme toggle.
- **Responsive Design**: Works seamlessly on desktop and mobile.

## Prerequisites

You need **Node.js** installed on your machine to run this project.
Download it here: [https://nodejs.org/](https://nodejs.org/)

## Getting Started

1.  **Install Dependencies**
    Open your terminal in the project directory and run:
    ```bash
    npm install
    ```

2.  **Run the Development Server**
    Start the app locally:
    ```bash
    npm run dev
    ```
   

3.  **Build for Production**
    To create a production build:
    ```bash
    npm run build
    ```

## Project Structure

- `src/components`: Reusable UI components and Layouts.
- `src/pages`: Main application views (Dashboard, Employees, Tasks).
- `src/context`: State management (TaskContext, ThemeContext).
- `src/data`: Mock data for the application.
